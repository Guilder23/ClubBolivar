PREREQUISITOS
═══════════════════════════════════════════════════════════════════

ANTES de ejecutar, necesitas:

1. ✓ Base de datos MySQL creada: "club_bolivar"
   
   mysql -u root -p
   CREATE DATABASE club_bolivar;
   USE club_bolivar;
   SOURCE database.sql;

2. ✓ Editar config/database.php con tus credenciales:
   const DB_USER = 'root';
   const DB_PASS = 'tu_contraseña';
   const DB_NAME = 'club_bolivar';

3. ✓ PHP 7.4+ instalado

4. ✓ MySQL Server corriendo

SI QUIERES PROBAR SOLO LA CONEXIÓN
═══════════════════════════════════════════════════════════════════

Crea un archivo test.php en ClubBolivar/:

<?php
require 'config/database.php';

if ($conn->connect_error) {
    echo "❌ Error de conexión: " . $conn->connect_error;
} else {
    echo "✓ Conexión exitosa!";
    echo "<br>✓ Base de datos: " . DB_NAME;
}
?>

Luego ejecuta:
   php -S localhost:8000
   
Y abre:
   http://localhost:8000/test.php
